# Regression checklist — Search (Chrome, desktop)

## Search by text
- [ ] Поиск доступен в шапке на всех страницах
- [ ] Ввод запроса + Enter → отображается выдача
- [ ] Клик по “лупе” → отображается выдача
- [ ] Пустой запрос → поиск не выполняется
- [ ] Отображается “Найдено товаров: X”
- [ ] Карточки товаров отображаются (минимум: фото/название/цена)

## Tags / уточнения
- [ ] При наличии уточняющих тегов они отображаются
- [ ] Теги кликабельны
- [ ] Клик по тегу обновляет выдачу
- [ ] Если уточнений нет — теги не показываются

## Empty state
- [ ] При 0 результатов: “Ничего не найдено”
- [ ] Есть кнопка “Перейти в каталог”, ведёт в каталог

## Photo search (AI)
- [ ] Блок “Найдите запчасти по фото” кликабелен и открывает форму
- [ ] Обязательное поле: файл (Найдите запчасти по фото)
- [ ] “Отправить” disabled до загрузки файла
- [ ] Допустимые форматы: JPG/JPEG, PNG
- [ ] Файл > 10 МБ → понятная ошибка, отправка недоступна
- [ ] Неподдерживаемый формат → понятная ошибка, отправка недоступна
- [ ] При отправке отображается loader/индикатор обработки
- [ ] Успех: отображается выдача “похожие запчасти” (карточки)
- [ ] Нет совпадений: отображается “Ничего не найдено” (или эквивалент)
- [ ] Ошибка обработки: “Не удалось обработать изображение…” + возможность повторить
