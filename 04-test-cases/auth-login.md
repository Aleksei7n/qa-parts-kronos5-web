# AUTH

## Login (Email only)

**TC-AUTH-001**  
Title: Открытие/закрытие модального окна “Войти” (все способы)  
US: US-LOGIN-1  
Module: Auth  
Type: UI  
Priority: P1  
Severity: Minor  

Preconditions:
- Пользователь не авторизован.
- Открыта главная страница сайта.

Test Data: —  

Steps:
1) На главной странице найти элемент входа (кнопка/ссылка “Войти” или иконка профиля) и нажать на него.  
2) Убедиться, что открылась модальная форма “Войти” и видимы поля **Email** и **Пароль**, а также ссылки **“Регистрация”** и **“Забыли пароль? Восстановить”**.  
3) Нажать на крестик закрытия в правом верхнем углу модального окна.  
4) Повторно открыть модалку “Войти” тем же способом.  
5) Нажать клавишу **Esc**.  
6) Повторно открыть модалку “Войти”.  
7) Кликнуть мышью вне модального окна (по затемненной области/оверлею).  

Expected Result:
- Модальное окно “Войти” открывается при нажатии на элемент входа.  
- Модалка корректно закрывается крестиком, клавишей Esc и кликом по оверлею.  
- После закрытия не остается затемнения/оверлея; элементы страницы кликабельны.  

Postconditions: —

**TC-AUTH-005**  
Title: Неверные учетные данные — отображается “Неверный логин или пароль”, вход не выполняется  
US: US-LOGIN-1  
Module: Auth  
Type: Negative  
Priority: P0

- Открыта модалка “Войти”.

**TC-AUTH-007**  
Title: Ссылки “Регистрация” и “Забыли пароль? Восстановить” доступны и открывают соответствующие формы  
US: US-LOGIN-1  
Module: Auth  
Type: UI  
Priority: P2  
Severity: Minor  

Preconditions:
- Пользователь не авторизован.
- Открыта модалка “Войти”.

Test Data: —  

Steps:
1) Не заполняя поля Email и Пароль, проверить наличие ссылки “Регистрация”.  
2) Нажать “Регистрация” и убедиться, что открылась форма регистрации.  
3) Закрыть регистрацию (крестик/кнопка закрытия/клик по оверлею).  
4) Вернуться к модалке “Войти” (если требуется — открыть снова).  
5) Нажать “Забыли пароль? Восстановить”.  
6) Убедиться, что открылась форма восстановления пароля.  

Expected Result:
- Ссылка “Регистрация” видима и кликабельна; открывает форму регистрации.  
- Ссылка “Забыли пароль? Восстановить” видима и кликабельна; открывает форму восстановления.  
- Для открытия ссылок не требуется предварительный ввод данных и не зависит от ошибки логина/пароля.  

Postconditions: —
